Regras do Jogo Trilha

O Jogo Trilha é um jogo de tabuleiro de
n x n x n (n = {2,3,4,5})
pontos conectados por linhas. O objetivo é formar "moinhos", que são três peças do mesmo jogador em uma linha reta (não contam diagonais).

  • Criar o maior número possível de "moinhos".
  • Eliminar as peças do oponente ao completar um "moinho".
  • O jogo termina quando um jogador não tem mais peças ou movimentos válidos.

Fases do Jogo:

  1. Fase 1: Colocar as peças no tabuleiro. Cada jogador coloca uma peça por turno.
  2. Fase 2: Movimentar as peças. Após todas as peças serem colocadas, os jogadores podem mover uma peça para uma casa adjacente.
  3. Fase 3: Fase de "voo". Se um jogador tiver apenas três peças, pode mover suas peças para qualquer posição no tabuleiro.

Formar um moinho permite remover uma peça do oponente. O jogo termina quando um jogador não pode mais fazer movimentos válidos.
